
# Honeybee Colony Stability Using IRT-FIS

An intelligent and non-invasive honeybee colony monitoring system using **Infrared Thermography (IRT)**, **Thermal Uniformity Index (TUI)**, and **Fuzzy Inference System (FIS)**.

This project analyzes thermal images to classify colony conditions into **Stable**, **Moderate**, and **Disturbed** states.

---

## 📌 Project Overview
Honeybee colonies regulate temperature for survival. This system uses **contactless thermal monitoring** to detect colony stability without disturbing bees.

---

## 🚀 Technologies Used
- Infrared Thermography (IRT)  
- Thermal Image Processing  
- Thermal Uniformity Index (TUI)  
- Fuzzy Logic Inference System (FIS)  

---

## 🔍 Features
- Thermal image analysis  
- Temperature extraction (Tmax, Tmin, Tmean)  
- TUI calculation  
- Fuzzy classification  
- Non-invasive monitoring  

---

## 🧠 Working Process
1. Capture thermal image  
2. Process image  
3. Extract temperature  
4. Compute TUI  
5. Classify colony  

---

## 📊 Results
- Tmax: 37.6°C  
- Tmin: 22.9°C  
- Tmean: 31°C  
- TUI: 0.89 – 0.95  

---

## 🌍 Applications
- Smart hive monitoring  
- Precision apiculture  
- Colony health analysis
